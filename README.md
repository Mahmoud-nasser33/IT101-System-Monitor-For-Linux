#System Monitor for Linux

A Bash script that monitors your linux system resources such as CPU, memory, disk usage, and running processes.

## Features

- Gui option (gnome)
- Supports manual or automatic monitoring modes
- Configurable refresh interval (time between updates)
----------
- Displays CPU usage percentage
- Shows memory usage
- Reports disk space usage
- Lists top running processes

## Requirements

- Linux operating system
- Bash shell
